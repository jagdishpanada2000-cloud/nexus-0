# Nexus App - Android (Flutter) 

A social media platform for **Gen-Z developers**, built for **Android** using **Flutter**.  
Users can **learn, build, collaborate, and grow**, with gamified quests and skill-based matching.  
Data is stored in **Firebase (JSON format)** for real-time updates.

## âš¡ Installation & Setup

1. Clone the repo:

```bash
git clone https://github.com/jagdishpanada2000-cloud/nexus-android.git
cd nexus-android
```

2. Install dependencies:

```bash
flutter pub get
```

3. Configure Firebase:

- Add your `google-services.json` to `android/app/`  
- Enable Firebase Authentication and Firestore  
- Set up your JSON structure in Firestore as per app requirements

4. Run the app:

```bash
flutter run
```
